About

This is a videogame name/concept generator. This is intended to help with game design practice, challenges or for inspiration in general. It helps getting you thinking in other directions. There is a total of 4942 words, which the generator can make into 6,326,616,080 different (though, possibly, very similar) game names.

To build this I used an old list of video games from Game Revolution, the NLTK library and the FreeBSD wordlist. Words were parsed from the list, classified and then cross-referenced against the wordlist. That gets you some random strings that mostly sound like videogame names. This blog post by Ian Schreiber got me thinking about the second part. The sentences for the mini-description are inspired by the MDA framework for game design.

Code available on GitHub.
